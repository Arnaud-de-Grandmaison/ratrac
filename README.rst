Build
=====

Building on a Mac can be done from the command line with:

.. code-block:: bash

  $ mkdir build
  $ cd build
  $ cmake -DCMAKE_BUILD_TYPE:STRING=Release -DCMAKE_EXPORT_COMPILE_COMMANDS:BOOL=ON -G Ninja ..
